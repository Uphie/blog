淅沥沥的秋雨 \
浅淡淡的回忆 \
寂寞蹲在这里 \
寻夏天的蚂蚁 \
长长的时间简简单单 \
铺出天上的那一片蓝 \
曾经开心的聊天 \
无际无边 \
寂寞涂成彩色 \
可快乐终被冷落 \
雕刻那个时刻 \
凝固你的笑靥 \
擦洗昨天的记忆 \
掩藏有你的身影 \
想解读那不能说的秘密

凝固你的笑靥 \
擦洗昨天的记忆 \

想解读那不能说的秘密 \
时间渐渐滑去 \
终于明白回忆是记忆不是你
