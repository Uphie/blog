# shell命令快捷键

## 编辑命令
```
# 移动光标到首部（即最左侧）
Ctrl+a

# 移动光标到尾部（即最右侧）
Ctrl+e

# 按字符右移（即前移）
# front
Ctrl+f

# 按字符左移（即后移）
# back
Ctrl+b

# 按单词右移
# front
Alt+f

# 按单词左移
# back
Alt+b

# 命令行首和光标间来回移动
Ctrl+xx

# 删除光标处的字符
Ctrl+d

# 删除光标前的字符
Ctrl+h

# 从光标处删除至命令行首（最左侧）
Ctrl+u

# 从光标处删除至命令行尾（最右侧）
Ctrl+k
```

## 重新执行命令
```
# 历史中的上一条命令
# previous
Ctrl + p

# 历史中的下一条命令
# next
Ctrl + n


# 逆向搜索命令历史
Ctrl + r

# 从历史搜索模式退出
Ctrl + g
```

## 控制命令

```
# 清屏
Ctrl + l

# 执行当前命令，并选择上一条命令
Ctrl + o

# 阻止屏幕输出
Ctrl + s

# 允许屏幕输出
Ctrl + q

# 终止命令
Ctrl + c

# 挂起命令
Ctrl + z
```
